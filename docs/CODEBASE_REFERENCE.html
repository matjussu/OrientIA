Détail par étape

S1
Scope Classifier
src/rag/scope_classifier.py
Regex prefiltres (urgent / identity / greeting) puis LLM Mistral Small JSON-mode si pas de match. Renvoie in_scope | out_of_scope | urgent | identity | greeting.
ScopeClassifier.classify()
0.5-1.5 s
S2
Intent + DomainHint
src/rag/intent.py
Classification rule-based : 7 intents (comparaison, geographic, realisme, ...) + 13 domain hints (crous, dares, apec, voie_pre_bac, ...). Drives top_k_sources + mmr_lambda.
classify_intent() + classify_domain_hint()
<10 ms
S3
RouterLLM + Retrieval
src/rag/retriever.py + router_llm.py + bm25_index.py
RouterLLM Mistral Small décide les sub-indexes ciblés (formations/metiers/aides/stats). FAISS dense top-150 (1024 dims). BM25 lexical RRF fusion optionnel.
RouterLLM.route() + retrieve_top_k() + BM25 RRF fusion
0.5-1.5 s
S4
Reranking
src/rag/reranker.py
Boosts multiplicatifs : parcoursup_rich 1.2, etab_named 1.1, niveau bac+5 1.15, domain-aware 1.3-1.5 si hint match. Calibration éprouvée Run F+G.
rerank()
<50 ms
S5
MMR diversification
src/rag/mmr.py
Maximal Marginal Relevance pour diversifier le top-K. λ piloté par intent (0.3 décou-verte, 0.9 conceptuel).
mmr_select()
<100 ms
S6
Golden QA few-shot
src/rag/pipeline.py::_maybe_build_golden_qa_prefix
Inject top-1 Q&A golden par similarité de ton (676 records multi-cat post-GQ rebuild). Lazy-load FAISS au 1er .answer().
_maybe_build_golden_qa_prefix()
0.2-0.5 s
S7
Generation Mistral
src/rag/generator.py + src/prompt/system.py
FactCard JSON structuré → Mistral medium max_tokens=800, temperature=0.3. Prompt v5 SPRINT11_P0 (4 directives anti-hallu).
generate() + SYSTEM_PROMPT_SPRINT11_P0
3-8 s
S8
Validator L1+L2+L3
src/rag/validator/
L1 rules regex (20+ patterns), L2 corpus_check cosinus seuil 0.55, L3 Mistral Small optionnel. honesty_score ∈ [0,1] + flagged boolean.
Validator.validate()
<10 ms (L1+L2), 2-4 s (L3 si actif)
S9
Retry conditionnel
src/rag/pipeline.py::_generate_with_retry
Si flagged + budget temps restant > 5s → 1 retry max avec domain hint injecté. Timeout wall-clock total 30s.
_generate_with_retry()
0 ou ~6 s
S10
Post-process
src/rag/post_process.py
strip_invented_urls, neutralize_broken_link_fallback (PR #135), fix_broken_markdown_tables, validate_onisep_slugs. Silently corrective.
post_process_answer()
<5 ms
